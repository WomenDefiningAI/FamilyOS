---

## Seasonal checklist

**Spring**
- [ ] HVAC filter change
- [ ] Test smoke and CO detectors
- [ ] Clean gutters
- [ ] Check exterior caulk and weatherstripping
- [ ] Check outdoor faucets after winter

**Summer**
- [ ] AC service if applicable
- [ ] Check sprinklers and irrigation
- [ ] Deck or patio inspection
- [ ] Clean dryer vent

**Fall**
- [ ] HVAC filter change
- [ ] Heating system check before first cold snap
- [ ] Clean gutters after leaves
- [ ] Check insulation and drafts
- [ ] Test sump pump if applicable

**Winter**
- [ ] Pipe insulation check
- [ ] Stock emergency kit (flashlights, batteries, water)
- [ ] Check roof after heavy snow
- [ ] Stock ice melt
